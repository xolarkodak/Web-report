Способи функціонального застосування JavaScript

JavaScript можна використовувати в HTML для створення
динамічних та інтерактивних веб-сторінок. Ось кілька способів, як це можна зробити:

Вбудований JavaScript: Можна включати JavaScript код прямо всередині HTML документу, використовуючи тег <script>.

<script>
              // JavaScript код тут
          </script>

Зовнішній JavaScript файл: Іншим способом є включення зовнішнього файлу JavaScript за допомогою атрибуту src тегу <script>. Це дозволяє тримати JavaScript код окремо від HTML.

<script src="script.js"></script>

Опис обробників подій: Можна додавати обробники подій безпосередньо в HTML елементи за допомогою атрибутів, таких як onclick, onmouseover, тощо.

<button onclick="myFunction()">Натисни мене</button>

Зміна HTML вмісту: JavaScript може змінювати HTML вміст, додаючи, видаляючи або змінюючи елементи та їх атрибути.

<div id="demo">Це початковий текст</div>
          <script>
              document.getElementById("demo").innerHTML = "Це новий текст";
          </script>

Маніпулювання стилями CSS: JavaScript може змінювати стилі елементів HTML, що дозволяє створювати анімації та змінювати вигляд сторінки динамічно.

<button onclick="changeColor()">Змінити колір</button>
          <script>
              function changeColor() {
                  document.body.style.backgroundColor = "lightblue";
              }
          </script>

Використання AJAX для асинхронного завантаження даних: JavaScript може використовувати AJAX для асинхронного взаємодії з сервером, без перезавантаження сторінки.

<button onclick="loadData()">Завантажити дані</button>
          <div id="result"></div>
          <script>
              function loadData() {
                  var xhr = new XMLHttpRequest();
                  xhr.open("GET", "data.txt", true);
                  xhr.onreadystatechange = function() {
                      if (xhr.readyState == 4 && xhr.status == 200) {
                          document.getElementById("result").innerHTML = xhr.responseText;
                      }
                  };
                  xhr.send();
              }
          </script>
