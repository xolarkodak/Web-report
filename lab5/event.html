Події в JavaScript

Події в JavaScript - це спосіб реагування на дії користувача або на інші події, що відбуваються у веб-додатку. Коли відбувається певна дія, така як клік миші, натискання клавіші або завантаження сторінки, виникає подія, на яку можна зв'язати функцію обробника. Функція обробника виконується у відповідь на подію.

Наприклад, реагуючи на клік кнопки, можна викликати функцію, яка змінює вміст сторінки або виконує інші дії. Ось простий приклад:


            <body>
                <button id="myButton">Натисни мене</button>
                <p id="message"></p>
    
                <script>
                    // Отримуємо кнопку за її id
                    var button = document.getElementById('myButton');
    
                    // Додаємо обробник події click
                    button.addEventListener('click', function() {
                        // Змінюємо вміст елемента з id "message"
                        document.getElementById('message').textContent = 'Кнопка була натиснута!';
                    });
                </script>
            </body>
