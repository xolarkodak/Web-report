Схеми URL в JavaScript

Схеми URL в JavaScript дозволяють працювати з URL-адресами, розбираючи їх на складові частини, такі як протокол, хост, шлях, параметри та фрагменти. Ось деякі основні методи та властивості, які можна використовувати для роботи з URL-адресами в JavaScript:

  1. URL: Це вбудований об'єкт, який представляє URL-адресу. Він має властивості, такі як href, protocol, hostname, pathname, search та інші, які дозволяють отримувати різні частини URL. 
    
let url = new URL('https://example.com/path?query=value');
console.log(url.hostname); // виведе "example.com"
console.log(url.search);   // виведе "?query=value"
  2. URLSearchParams: Цей об'єкт дозволяє працювати з параметрами запиту в URL. 
    
let params = new URLSearchParams('key1=value1&key2=value2');
console.log(params.get('key1')); // виведе "value1"
  3. encodeURIComponent та decodeURIComponent: Ці функції використовуються для кодування та декодування компонентів URL. 
    
let encoded = encodeURIComponent('шлях з пробілами');
console.log(encoded); // виведе "%D1%88%D0%BB%D1%8F%D1%85%20%D0%B7%20%D0%BF%D1%80%D0%BE%D0%B1%D1%96%D0%BB%D0%B0%D0%BC%D0%B8"
let decoded = decodeURIComponent(encoded);
console.log(decoded); // виведе "шлях з пробілами"

Використання цих методів дозволяє зручно працювати з URL-адресами в JavaScript, що є важливим для багатьох веб-застосунків та скриптів.
