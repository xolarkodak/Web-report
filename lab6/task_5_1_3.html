Завдання №3:

З об'єкту medicines потрібно отримати масив в якому будуть лише назви препаратів. З масиву потрібно прибрати медикамент и, в яких строк зберігання уже пройшов . У новому масиві відсортувати медикаменти у хронологічному порядку. Результат вивести у консоль. Застосувати стрілочні функції

const medicines = {
Агалгін: new Date("2022-05-01"),
Ношпа: new Date("2025-07-02"),
Альфахолін: new Date("2024-12-21"),
Аспірин: new Date("2022-08-15"),
Аспаркам: new Date("2024-04-18"),
};

Код JavaScript:

const medicines = { Агалгін: new Date("2022-05-01"), Ношпа: new Date("2025-07-02"), Альфахолін: new Date("2024-12-21"), Аспірин: new Date("2022-08-15"), Аспаркам: new Date("2024-04-18"), }; // Отримуємо поточну дату const currentDate = new Date(); // Створюємо масив з назвами медикаментів, строк зберігання яких ще не пройшов const validMedicines = Object.entries(medicines) .filter(([name, expiryDate]) => expiryDate > currentDate) .sort((a, b) => a[1] - b[1]) .map(([name, expiryDate]) => name); console.log(validMedicines);
