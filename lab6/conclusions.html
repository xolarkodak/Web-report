Висновки :

В ході виконання лабораторної роботи :

Класи: Класи в JavaScript є основою об’єктно-орієнтованого програмування.
Вони дозволяють створювати об’єкти з властивостями та методами. Класи можуть
успадковувати властивості та методи від інших класів через механізм наслідування.

Об’єктна модель документа (DOM): DOM дозволяє взаємодіяти з HTML та XML документами.
За допомогою DOM, JavaScript може отримувати доступ до та змінювати елементи, атрибути та текст в документі.

Події: Події в JavaScript - це дії, які відбуваються в браузері, наприклад, клік миші,
завантаження сторінки, натискання клавіші тощо. JavaScript може “слухати” ці події та
виконувати код відповідно до них.

Об’єкт події: Коли подія відбувається, браузер створює об’єкт події, який містить
інформацію про подію. Цей об’єкт передається як аргумент до обробника подій.

Делегування подій: Делегування подій - це техніка, яка дозволяє нам обробляти події
на високому рівні (наприклад, на рівні document), замість того, щоб назначати обробник
подій кожному окремому елементу.

Throttle: Throttling - це техніка оптимізації, яка обмежує кількість разів, коли функція
може виконуватися протягом певного періоду часу. Це корисно для подій, які відбуваються
дуже часто, наприклад, прокрутка або зміна розміру вікна.

Модульність коду: Модульність коду означає розбиття коду на незалежні частини або модулі. Це полегшує розробку, тестування та зберігання коду.
