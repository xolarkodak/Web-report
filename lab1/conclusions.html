Висновки :

В ході виконання лабораторної роботи :

  • 1. Обрано предметну галузь для майбутніх проектів, що дозволило визначити цільову аудиторію та основні вимоги до сайту.
  • 2. Створено HTML код базової сторінки звітного документу, що є основою для подальшого розвитку проекту.
  • 3. За допомогою каскадних таблиць стилів (CSS) було стилізовано елементи сторінки, що покращило її візуальну привабливість та зручність користування.
  • 4. Вивчено та додано на сторінку такі елементи, як:
    • Посилання, які забезпечують навігацію між різними частинами сайту та зовнішніми ресурсами.
    • Таблиці, які структуровано відображають дані та інформацію.
    • Форми, що дозволяють користувачам вводити та надсилати інформацію.
    • Зображення, які візуально доповнюють контент сторінки.
    • Списки, що допомагають структурувати інформацію у вигляді пунктів.
    • Заголовки різних рівнів, що забезпечують логічну організацію та ієрархію контенту
  • 5. Використовуючи сервіс GitHub Pages, сайт було розміщено в Інтернеті, що забезпечило доступ до нього з будь-якого місця з підключенням до мережі.
  • Таким чином, під час лабораторної роботи було закладено основи для створення повноцінного веб-проекту, включаючи вибір тематики, розробку HTML структури, застосування CSS для стилізації, інтеграцію важливих веб-елементів та публікацію сайту в Інтернеті. Це забезпечує фундаментальні знання та навички для подальшого розвитку та вдосконалення веб-проектів.
