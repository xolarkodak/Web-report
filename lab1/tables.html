Таблиці :

Елементи, які можуть або повинні використовуватись в таблицях:

Назва елементу Його значення
<caption> Тег визначає заголовок (назву) таблиці. Заголовок таблиці має розміщуватись відразу після відкриваючого тега, тобто після <caption>. Текст котрий знаходиться в елементі <caption> відображається перед таблицею і описує її зміст.
<colgroup> Цей елемент дозволяє зменшити код таблиці за рахунок скорочення атрибутів, що повторюються, і при його наявності браузер починає показувати вміст таблиці, не чекаючи її повного завантаження.
<thead> Використовується для зберігання групи заголовків HTML-таблиці (верхній колонтитул), котрі будуть відображені зверху таблиці.
<tbody> Визначає тіло HTML-таблиці.
<tr> Призначений для створення одного рядка таблиці.
<td> Призначений для створення однієї комірки таблиці (звичайну комірку з даними).
<th> Призначений для створення однієї комірки-заголовка таблиці.
<tfoot> Використовується для групування підсумкових рядків в HTML-таблиці, котрі будуть відображені внизу таблиці.
